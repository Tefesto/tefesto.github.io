# Yt-dlp

> Програма для скачування відео або аудіо з Ютубу.

## Встановлення

`winget install yt-dlp.yt-dlp`  

Разом з програмою має встановитися конвертер ffmpeg.  
Якщо ffmpeg не встановився, потрібно встановити його самостійно.  

`winget search ffmpeg`  
`winget install Gyan.FFmpeg`
